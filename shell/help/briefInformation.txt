Как использовать программу:
- Изображения хранятся в папек pics.
- Изображения обратной стороны хранятся в папке pics/back
- Файл изображения каждой карточки должен иметь имя X.jpg,
где X - это номер строки в exel-фале, начиная с 2.jpg
- Все данные по карточкам хранятся в exel-фале в папке slu в файле data
- В файле data хранится информация по каждой карточке в отдельной строке
- В первой строке находятся названия данных,
в последующих данные по каждой карточке построчно
- Программа считывает данные построчно,
включая пустые строки. БУДЬТЕ ВНИМАТЕЛЬНЫ!
