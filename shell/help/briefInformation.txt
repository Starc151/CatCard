Как пользоваться программой.
Основные тезисы:
- Приложение вместе с папками slu и pics
должно находиться в одной папке.
- В папке slu находится exel-файл data.xlsx, с данными каталогов.
- Каждый лист файла относится к оттдельглму каталогу
- В первой строке листа находится название поля
- В далее построчно идут данные по каждой карточке
- Изображения хранятся в папке pics/название каталога
- Имя папки должно совпадать с названием каталога в exel-файле,
иначе прилжение не найдёт нужное изображение!
- Изображения обратной стороны карточкт хранятся в папке
pics/название каталога/back
- Файл изображения каждой карточки должен иметь имя X.jpg,
где X - это номер строки в exel-файле, начиная с 2.jpg
- Программа считывает данные построчно,
включая пустые строки. БУДЬТЕ ВНИМАТЕЛЬНЫ!

Для более подробной справкой запустите файл HELP.EXE (в разработрке)
